## What I Learned

During this internship task, I learned the basics of HTML, CSS, and GitHub. I created a personal bio page, built a portfolio page, and learned how to structure web pages using headings, paragraphs, lists, cards, and links. I also understood how to use Git and GitHub for version control, including creating repositories, branches, commits, pull requests, and merging changes into the main branch.

## What Was Hard

The most challenging part was working with GitHub branches and pull requests. At first, I created my portfolio page directly in the main branch, which made it difficult to create a pull request because there were no differences between the branches. I learned how branches work and how important it is to make changes in the feature branch before creating a pull request. Understanding Git commands and the GitHub interface also took some practice.

## What I Want to Learn Next

Next, I want to improve my web development skills by learning JavaScript to make websites interactive. I also want to learn responsive web design using CSS so that websites look good on all devices. In addition, I hope to explore React, APIs, and advanced Git features to build more professional projects and collaborate effectively with others.
